public class SimplePieceAdjuster
extends Object
implements ChineseCharacterTypeAdjuster

區塊活字調整工具。調整PieceMovableType，因為把活字的資訊全部集中在同一個物件上（ Piece， RectangularArea型態 ），方便函式傳遞與使用，而且物件上也有相對應操縱的函式。

SimplePiece是在設定時兩兩配對後定框，調整時更改部件大小，但無法物件難實作距離貼近或拉開。
