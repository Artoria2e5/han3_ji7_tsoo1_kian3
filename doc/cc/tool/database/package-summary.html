Package cc.tool.database

資料庫相關處理工具，將有漢字構形資料庫從微軟Access、到pg資料庫，再移植成漢字組建資料庫的工具，以及相關驗證程式。

PgsqlConnection Pgsql連線物件。

全字庫大五碼對照表輸入資料庫 將大五碼（Big5）輸入至資料庫的型態。

全字庫對照表輸入資料庫 協助全字庫對照表資料輸入資料庫的模組。
