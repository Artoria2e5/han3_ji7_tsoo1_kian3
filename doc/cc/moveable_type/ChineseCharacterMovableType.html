public abstract class ChineseCharacterMovableType
extends Object
implements 漢字組建活字

儲存漢字活字樹狀結構。「獨體為文，合體為字」，樹狀結構中的葉子為文，其他上層節點為字。 ChineseCharacterMovableType為 ChineseCharacterMovableTypeWen及 ChineseCharacterMovableTypeTzu 的共用介面，方便以後活字的調整。
