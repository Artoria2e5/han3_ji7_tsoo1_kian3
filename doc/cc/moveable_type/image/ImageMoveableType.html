public interface ImageMoveableType
extends 漢字組建活字

圖片活字的介面。ImageMoveableType 在列印時才讀取字體，處理時不考慮部件的差異，皆視為同樣大小，每個部件當做矩形，記錄預計長寬（region）及位置（ position），目前長寬（scaler ）是給合體活字儲存紀錄的。
